# 🌐 Personal Portfolio

A modern and responsive personal portfolio website built using **React**, **Vite**, and **Tailwind CSS** to showcase my projects, skills, and contact information.

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🟨 TypeScript

---

## 📂 Project Structure

```
portfolio/
├── src/
```

## 🧑‍💻 Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo.git
```

### 2. Navigate to project folder

```
cd your-repo
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```

---

## 📦 Build for Production

```
npm run build
```
